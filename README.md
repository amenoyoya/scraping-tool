# ScrapingTool

## 概要

スクレイピングを簡単に行うためのツール

## 開発環境

- 環境
  - Miniconda 4.5.11
  - Python 3.6.7
- ライブラリ
  - ブラウザ自動操作ライブラリ `selenium`
    ```sh
    pip install selenium
    ```
  - HTMLタグ解析ライブラリ `BeautifulSoup4`
    ```sh
    pip install beautifulsoup4
    ```
  - ElectronライクなGUIライブラリ `Eel`
    ```sh
    pip install eel
    ```
